# The Serene Haven Hotel - Simple Static Website

This project is a simple, static website for a fictional hotel named "The Serene Haven." It's designed to be straightforward, using pure HTML, CSS, and minimal JavaScript to showcase the hotel's features, rooms, amenities, and contact information.

## Project Structure

-   `index.html`: The main entry point of the website, containing all the content and structure.
-   `css/style.css`: Contains all the styling rules for the website, including responsive design.
-   `js/script.js`: Provides basic interactivity, such as smooth scrolling for navigation links and highlighting the active section.
-   `favicon.ico`: A placeholder for the website's favicon (icon displayed in browser tabs).
-   `README.md`: This file, providing information about the project.

## Features

-   **Responsive Design**: Adapts to various screen sizes (desktop, tablet, mobile).
-   **Header & Navigation**: Sticky navigation bar with links to different sections.
-   **Hero Section**: A prominent welcome area with a background image.
-   **About Section**: Information about the hotel.
-   **Rooms Section**: Displays different room types with images, descriptions, and pricing.
-   **Amenities Section**: Lists the facilities and services offered by the hotel.
-   **Gallery Section**: Showcases hotel photos in a grid layout.
-   **Contact Section**: Provides contact details and a map placeholder.
-   **Footer**: Copyright information and social media links.
-   **Smooth Scrolling**: Enhanced user experience when navigating between sections.
-   **Active Navigation Highlight**: Highlights the current section in the navigation bar as the user scrolls.
